# SNAFOO
### Nerdery Snack Food Ordering System

## QuickStart:
1. Start a compatible server at ```localhost:3000```
1. Run ```npm i```
1. Run ```npm run start```

## Scripts:
Transpile and bundle: 
```
npm run build
```

Watch for file changes and bundle:
```
npm run watch
```

Start local app instance:
```
npm run start
```
